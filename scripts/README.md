# Scripts Directory

This directory contains all build, deployment, and utility scripts for the AI Message project.

## Files

- `build.bat` / `build.sh` - Build scripts for Windows/Unix
- `demo.bat` / `demo.sh` - Demo execution scripts
- `start.bat` / `start.sh` - Application startup scripts

## Usage

### Windows
```batch
# Build the project
scripts\build.bat

# Start demo
scripts\demo.bat

# Start application
scripts\start.bat
```

### Unix/Linux/macOS
```bash
# Build the project
scripts/build.sh

# Start demo
scripts/demo.sh

# Start application
scripts/start.sh
```
